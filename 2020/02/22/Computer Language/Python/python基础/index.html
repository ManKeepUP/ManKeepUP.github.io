变量和简单数据类型

变量

  在python中我们不需要事先指定类型就可使用一个变量(这一点和其他编程语言不同，比如C)，变量存储了一个与它相关联的信息。

命名和使用

  • 只能是字母、数字和下划线
  • 只能以字母或下划线开头
  • 不能含空格
  • 不能是关键字

变量名应既简短又具有描述性，慎用小写字母l和大写字母O，变量名避免使用大写字母。
